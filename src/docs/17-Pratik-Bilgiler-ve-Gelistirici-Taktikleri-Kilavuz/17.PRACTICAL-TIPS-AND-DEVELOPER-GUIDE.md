# JavaScript Rehberi – Temel ve İleri Seviye

Bu rehber JavaScript öğrenmek ve profesyonel seviyeye ulaşmak için hazırlanmış eksiksiz bir referanstır.

---

## Temel Seviye (Fundamentals)

JavaScript'in çekirdek yapılarını içerir. Tüm JS kullanıcıları için gereklidir.

### Değişken Tanımlama

```javascript
let isim = "Aylin";
const yas = 20;
var boy = 1.75;

console.log(isim, yas, boy);
```

---

### console.log() Kullanımı

```javascript
console.log("Merhaba Dünya");
console.log(`Merhaba ${isim}`);
```

---

### if / else if / else Kullanımı

```javascript
let notu = 85;

if (notu >= 90) {
  console.log("AA");
} else if (notu >= 70) {
  console.log("BB");
} else {
  console.log("CC");
}
```

---

### for Döngüsü

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}

let isimler = ["Ali", "Veli", "Ayşe"];
isimler.forEach(isim => console.log(isim));
```

---

### while Döngüsü

```javascript
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```

---

### Fonksiyon Tanımlama

```javascript
function selamVer(isim) {
  console.log(`Merhaba ${isim}`);
}

selamVer("Baris");
```

---

### Array Kullanımı

```javascript
let sayilar = [10, 20, 30];
sayilar.push(40);
console.log(sayilar);
console.log(sayilar[0]);
```

---

### Object Kullanımı

```javascript
let kisi = {
  isim: "Baris",
  yas: 30,
  meslek: "IT Admin"
};
console.log(kisi.isim);
```

---

### try / catch Kullanımı

```javascript
try {
  let sayi = parseInt(prompt("Sayı gir: "));
  console.log(sayi);
} catch (error) {
  console.log("Hatalı giriş");
}
```

---

### import / export Kullanımı

```javascript
// math.js
export const pi = 3.14;

// main.js
import { pi } from './math.js';
console.log(pi);
```

## Pratik Kullanım (Practical Usage)

### Tarayıcı Konsolu (Console) Ustalığı

* **$0 – $4:** Son tıklanan 5 elemente erişim.
* **copy(data):** Objeyi panoya kopyalar.
* **monitorEvents(element):** Olayları canlı izler.
* **Live Expressions:** Değişkeni sürekli takip et.

### JSX Cheat Sheet

| Özellik          | HTML               | JSX                      | Not                           |
| ---------------- | ------------------ | ------------------------ | ----------------------------- |
| Sınıf Tanımı     | class="box"        | className="box"          | `class` JS'de ayrılmış kelime |
| Satır İçi Stil   | style="color: red" | style={{ color: 'red' }} | Çift süslü parantez           |
| Etiket Kapatma   | <img>              | <img />                  | Tüm etiketler kapatılmalı     |
| JS Değişkeni     | Kullanılmaz        | {degiskenAdi}            | JS kodu yazılabilir           |
| Olay Yönetimi    | onclick="fn()"     | onClick={fn}             | CamelCase + referans          |
| Label Bağlantısı | for="inputID"      | htmlFor="inputID"        | for ayrılmış                  |

### JSX Kuralları

* Tek bir ana eleman döndür: `<> ... </>`
* Koşullu render: `{isLoggedIn && <button>Çıkış Yap</button>}`
* Liste map ile dön: `{items.map(item => <li key={item.id}>{item.name}</li>)}`

## İleri Seviye (Advanced)

### Class (OOP)

```javascript
class Kisi {
  constructor(isim) {
    this.isim = isim;
  }
  selam() {
    console.log(`Merhaba ${this.isim}`);
  }
}

let k = new Kisi("Baris");
k.selam();
```

### API ve Fetch

```javascript
fetch('https://api.github.com')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
```

### JSON İşlemleri

```javascript
let data = '{"isim":"Baris"}';
let obj = JSON.parse(data);
console.log(obj.isim);
```
